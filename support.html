FAQ (よくある質問)

Q: krmr Clientを入れるとクラッシュします / 起動しません。

A: 主な原因と確認事項です:

  • 前提Modの不足/バージョン違い: 導入手順にある通り、正しいバージョンの Fabric Loader と **Fabric API** (Ver X.X.X以上) が `mods` フォルダに導入されていますか？ Fabric APIの入れ忘れに注意してください。
  • Minecraftバージョンの不適合: krmr Client v1.0 は Minecraft 1.XX.X 用です。異なるバージョンで起動しようとしていませんか？
  • 他のModとの競合: 他に入れているModがある場合、競合の可能性があります。トラブルシューティングの競合特定手順を試してください。
  • インストール失敗: インストーラー(`krmr-Installer.jar`)は正しく完了しましたか？再度試してみてください。
解決しない場合は、コミュニティ (Discord推奨) でクラッシュレポート (`.minecraft/crash-reports` 内の .txt ファイル) を共有してください。

Q: 設定画面はどうやって開きますか？

A: ゲーム内でデフォルトで 右Shiftキー を押すと krmr Client の設定画面が開きます。キーバインドは設定画面内で変更可能です。Mod ModMenu を導入していれば、そこからもアクセスできます。

Q: Forge版はありますか？ / 対応バージョンを増やせますか？

A: 現在 krmr Client は Fabric Mod Loader のみ対応しており、Forge版のリリース予定はありません。ダウンロードページに記載の Minecraft バージョンのみサポートしています。古いバージョンや将来のバージョンへの対応は未定です。

Q: 特定の機能（例: フルブライト）を使ってもBANされませんか？

A: krmr Client の一部機能は、サーバーによってはルール違反とみなされる可能性があります。特にマルチプレイヤーサーバーで使用する場合は、必ず事前にサーバーのルールを確認してください。ルールの範囲内で自己責任での利用をお願いします。開発チームはサーバーでのアカウントBAN等に関して一切責任を負いません。

Q: FPSがあまり上がりません。

A: パフォーマンス改善の効果は環境に依存します。Minecraft自体のビデオ設定を見直す、他のModとの組み合わせを確認する、krmr Client内の関連設定を調整するなどを試してください。詳細はFAQ: パフォーマンスが改善されない場合の詳細 や トラブルシューティング をご覧いただくか、コミュニティでご相談ください。

Q: バグを見つけた / 機能をリクエストしたい。

A: ありがとうございます！ コミュニティ (Discord または GitHub Issues) にて報告をお願いします。バグの場合は、再現方法や環境、エラーログなどを詳しく記載いただけると助かります。

トラブルシューティング

1. クラッシュ / 起動失敗

  • まず確認: FAQ のクラッシュに関する項目 を確認。特に前提Mod (Fabric API) とMinecraftバージョンの整合性。
  • クラッシュレポート: .minecraft/crash-reports内の最新.txtファイルを確認。原因究明の手がかりになります。
  • Mod競合特定: krmr Client と Fabric API 以外のModを一度全て外し、起動するか確認。起動した場合、外したModを少しずつ戻して原因を特定（二分探索が有効）。
  • 設定リセット: .minecraft/config内のkrmr Client関連の設定ファイルを削除/リネームして起動を試す。（ファイル名は例: `krmrclient.json`など mod実装による）
  • 再インストール: Fabric Loader, Fabric API, krmr Client (インストーラー経由) を一度すべてアンインストールし、手順通りに再インストールしてみる。

2. 特定機能の問題 / 表示不具合

  • 設定確認: 設定画面（右Shift）で機能が有効か、関連オプションが意図通りか確認。
  • キー競合: 他のModやMinecraft本体とキーバインドが重複していないか確認、必要なら変更。
  • 互換性/シェーダー影響: 他の描画系Mod (Sodium等) やシェーダーパックを使用している場合、無効にして動作を確認。
  • 最新版確認: アップデート履歴 を見て、バグが修正されていないか確認。

上記で解決しない場合

問題の詳細（何をしてどうなったか、環境、エラーメッセージ全文、スクリーンショットなど）を添えて、公式Discordサーバー で質問してください。
