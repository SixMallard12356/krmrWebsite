導入手順 (Fabric)

1. Fabric Loader の準備

  1. Minecraft Java Edition をインストールしていることを確認します。
  2. Fabric Loader 公式サイトから **Installer (.jar)** をダウンロードします。
  3. ダウンロードした Fabric Installer を実行します。
  4. 「Client」タブで、「Minecraft Version」に krmr Client が対応している **Minecraft バージョン (1.XX.X)** を選択します。(※必ず正しいバージョンを指定してください！) Loader Version は最新を選択し、「Install」をクリックします。
  5. "Successfully installed..." と表示されたら成功です。
  6. Minecraft Launcher を起動し、「起動構成」タブに "fabric-loader-1.XX.X" というプロファイルが作成されていることを確認します。
  7. 一度、作成した Fabric プロファイルで Minecraft を起動し、タイトル画面まで進んだら終了します。（必要なフォルダが生成されます）

2. krmr Client のインストール

  1. 上でダウンロードしたインストーラー krmr-Installer.jar を実行（ダブルクリック）します。
  2. インストーラーの指示に従い、インストール先の Minecraft ディレクトリ (.minecraft) が正しいことを確認して「Install」を実行します。
  3. "Installation successful" のようなメッセージが表示されれば完了です。
  4. (必須)前提Mod: このクライアントは Fabric API (バージョン X.X.X 以上 - ※必ず対応バージョンを指定！) を必要とします。対応するバージョンのFabric APIの `.jar` ファイルをダウンロードしてください。
  5. OS に応じて Minecraft の mods フォルダを開きます:
    • Windows: %appdata%\.minecraft\mods
    • macOS: ~/Library/Application Support/minecraft/mods
    • Linux: ~/.minecraft/mods
    ※ `mods` フォルダがない場合は作成してください。
  6. ダウンロードした **Fabric API の `.jar` ファイル** を、この `mods` フォルダの中に入れます。（krmr Client 本体はインストーラーにより適切な場所に配置されるため、modsフォルダに入れる必要はありません。）

3. 起動と確認

  1. Minecraft Launcher を起動します。
  2. 起動構成で Fabric プロファイル (例: "fabric-loader-1.XX.X") が選択されていることを確認します。
  3. 「プレイ」ボタンでゲームを起動します。
  4. タイトル画面に "krmr Client" の表示が出れば成功です。エラーが出る場合は トラブルシューティング へ。
  5. ゲーム内でデフォルト設定キー [右Shiftキー] を押して krmr Client の設定画面が開くか確認してください。

システム要件:

  • Minecraft Java Edition: バージョン 1.XX.X (※必ず対応バージョンを指定！)
  • Fabric Loader
  • Fabric API: バージョン X.X.X 以上 (※必ず対応バージョンを指定！)
  • Java: Minecraft が要求するバージョン
  • RAM: 最低 4GB 推奨
